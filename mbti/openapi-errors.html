错误码说明

本文档详细列出了 MBTI API 接口在调用过程中可能返回的状态码及其具体含义，帮助您在集成过程中快速定位并解决问题。

01. 通用状态码

状态码 含义 详细描述与建议
200 SUCCESS 请求成功。业务数据包含在返回体的 data 字段中。
400 BAD_REQUEST 请求参数错误。请检查 JSON 格式是否正确或必填参数是否缺失。
401 UNAUTHORIZED 身份验证失败。API Key 无效、已禁用或已过期。
403 FORBIDDEN 权限不足。您当前的套餐级别不支持访问该特定 API 终结点。
500 INTERNAL_ERROR 服务器内部错误。请记录 Request ID 并联系我们的技术支持团队。

02. 签名校验相关错误

lock_open

注意：安全验证

为了保证数据传输的安全，所有写入类请求均需要通过 MD5 签名校验。若签名验证失败，接口将返回以下特定错误码。

错误码 具体原因 故障排除建议
40101 签名不匹配 生成的签名值与服务器计算结果不一致。请检查待签名字符串的排序顺序及 Secret Key。
40102 重放攻击检测 Timestamp 字段与服务器当前时间偏差超过 300 秒。请同步您的系统时钟。
40103 Nonce 重复 Nonce 随机字符串已在近期被使用。请确保每次请求的 Nonce 具有唯一性。
